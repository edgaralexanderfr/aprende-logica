La lógica
A la lógica la podemos considerar desde dos puntos de vista; como ciencia y como técnica. Como ciencia porque investiga, establece y desarrolla los métodos y principios aptos para distinguir el razonamiento correcto del incorrecto. Como técnica, la lógica permite desarrollar la destreza para interpretar el razonamiento incorrecto.
La lógica es el estudio de los métodos y principios que se usan para distinguir el razonamiento bueno (correcto) del malo (incorrecto). Esta definición no implica que sólo el estudiante de lógica pueda razonar bien o correctamente. También estudia los métodos y las leyes que permiten distinguir un razonamiento valido de un razonamiento no valido.

En este apartado se estudiará la lógica proposicional también llamada Lógica Simbólica o Matemática es un sistema formal, diseñado para analizar ciertos tipos de argumentos y símbolos utilizados en la formación de nuevos argumentos que podrían ser verdaderos o falsos. La lógica como sistema formal, trata de explicar la comprensión de la noción de consecuencia lógica para el nivel de argumentos que se estudia.
El razonamiento lógico se emplea en matemáticas para demostrar teoremas; en ciencias de la computación e informática para verificar si son o no correctos los programas; en las ciencias física y naturales, para sacar conclusiones de experimentos; y en las ciencias sociales y en la vida cotidiana, para resolver una multitud de problemas. Ciertamente se usa en forma constante el razonamiento lógico para realizar cualquier actividad.
